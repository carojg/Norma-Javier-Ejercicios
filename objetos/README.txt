Procedimiento para ejecutar el programa:
1. Ejecutar el archivo llamado "Objetos.java"
2. Al ejecutarse te pedirá que introduzcas la medida de un lado.
3. Aparece un menu donde se selecciona entre las opciones disponibles
4. Una vez seleccionada la opción se muestra el resultado y se elige si regresar al menu o capturar otro dato.
